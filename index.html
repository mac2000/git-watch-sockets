Git Watch

Listen for webhooks and promote them via websocket

JWT is required and can be obtained here

Usage example:

			const token = '...'
			const socket = new WebSocket(`ws://localhost:3000?access_token=${token}`)
			socket.onmessage = event => console.log(JSON.parse(event.data))

Do not forget to replace localhost:3000 to correct value

Notes

  • App should be watching and refreshing token if needed, git watch itself does not know nothing about token server
  • App should be ready for connection loose e.g. new version is deployed or connectivity issues
